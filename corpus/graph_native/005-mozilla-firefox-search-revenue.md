# Mozilla, Firefox, and search default revenue

Source: https://www.mozilla.org/en-US/foundation/annualreport/2022/
Source: https://www.justice.gov/opa/pr/justice-department-statements-us-v-google-search-antitrust-decision

Mozilla develops the Firefox browser. Mozilla's public financial reports describe
royalties from search partnerships as a major revenue source. In the Google search
antitrust case, browser default agreements were part of the competitive dispute.
Firefox historically offered Google Search as a default search option in many
markets, and Mozilla received search-related royalty revenue. This made Mozilla a
smaller but important browser actor in the same default-search ecosystem that also
included Apple Safari and Microsoft Bing.

Relations:

- Mozilla -> develops -> Firefox
- Firefox -> browser by -> Mozilla
- Mozilla -> receives royalties from -> search partnerships
- Google Search -> default search option in -> Firefox
- Google -> search partner of -> Mozilla
- Firefox -> competes with -> Safari
- Safari -> developed by -> Apple
- Bing -> search engine operated by -> Microsoft
- Department of Justice -> examined -> browser default agreements
- browser default agreements -> affect distribution of -> Google Search
