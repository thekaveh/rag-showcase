# Microsoft, OpenAI, Azure, and investment

Source: https://blogs.microsoft.com/blog/2023/01/23/microsoftandopenaiextendpartnership/
Source: https://openai.com/index/openai-and-microsoft/

Microsoft and OpenAI expanded their partnership in 2023 through a multiyear,
multibillion-dollar investment. Microsoft Azure became OpenAI's exclusive cloud
provider for OpenAI workloads, APIs, products, and research. Microsoft integrated
OpenAI models into products including GitHub Copilot, Microsoft 365 Copilot, Bing,
and Azure OpenAI Service. OpenAI used Azure infrastructure to train and deploy
large AI systems. The partnership connected OpenAI's model development to
Microsoft's cloud, developer, search, and productivity businesses.

Relations:

- Microsoft -> invested in -> OpenAI
- Microsoft Azure -> exclusive cloud provider for -> OpenAI
- OpenAI -> trains models on -> Microsoft Azure
- Azure OpenAI Service -> provides access to -> OpenAI models
- GitHub Copilot -> uses technology from -> OpenAI
- Microsoft 365 Copilot -> uses technology from -> OpenAI
- Bing -> integrated models from -> OpenAI
- Satya Nadella -> led -> Microsoft
- Sam Altman -> led -> OpenAI
- Microsoft -> commercializes -> OpenAI models
